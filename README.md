# Fissidens-600-project
Deep Learning and Foundation Models for Fine-grained Fissidens Classification


## Dataset

Fissidens-600

- 5 species
- 600 images
- curated from iNaturalist and field observations

## Models

- ResNet50
- Swin Transformer
- BioCLIP
- BioCLIP 2
- BioCLIP 2.5

## Methods

- Confusion Matrix
- t-SNE
- FIMF Score-CAM
